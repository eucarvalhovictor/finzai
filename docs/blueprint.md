# **App Name**: Lime Wallet

## Core Features:

- Financial Overview: Provides a comprehensive summary of the user's financial status, including balances, debts, and net worth.
- Credit Card Management: Enables users to track and manage their credit cards, including balances, transactions, and payment due dates.
- Investment Tracking: Allows users to monitor their investment portfolio, displaying performance metrics and asset allocation.
- Income and Expense Tracking: Facilitates detailed tracking of income and expenses with categorization for better analysis.
- AI-Powered Budgeting Tool: Suggests budget adjustments based on spending habits and financial goals using a generative AI model. This is a 'tool' in that it decides if there's information in your historical transaction that might affect the suggestion.

## Style Guidelines:

- Primary color: Lime green (#AEF200) to highlight key information and actions.
- Background color: Dark grey (#222222) to provide contrast and a modern feel.
- Accent color: Cyan (#00FFFF) for interactive elements and secondary actions.
- Body and headline font: 'Inter' sans-serif for clear readability and a modern aesthetic.
- Use minimalistic icons in lime green to represent categories and actions.
- A clean, card-based layout to present information in an organized and digestible manner.
- Subtle animations on data updates and transitions to enhance user engagement.
